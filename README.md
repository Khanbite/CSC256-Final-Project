# CSC256-Final-Project

## How to run
- First, have a working computer with JDK installed, anything after 23 would work.
- Either clone the repostory using the button on the top right. Press the url and copy it in the clone repository button on IntelliJ, then navigate to ChessGame.Java and run the file
- Or run the file through downloading a zip, extract and navigating over to ChessGame.java, and press run. since it is a java class compared to a java source file, everyone should be able to run
- If it is not compiled, run javac and run the .class file it generates, and then run the .class file
